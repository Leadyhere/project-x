# Current MVP Philosophy

The system is NOT:
- a therapist
- a diagnostic AI
- an emotional dependency system

The system IS:
- a listening-first conversational triage engine
- a psychological narrative understanding system
- a structured distress assessment platform
- a safety-oriented conversational assistant

Conversation follows:
- Motivational Interviewing inspired interaction
- minimal bot speaking
- user-led disclosure
- dynamic questioning
- contextual exploration

Risk classification is:
- deterministic
- rule-based
- auditable
- safety-first

LLMs are ONLY used for:
- conversational flow
- rephrasing
- reflective listening
- dynamic question generation

LLMs DO NOT decide:
- final risk
- escalation
- safety classification
